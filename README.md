## Install Backend
1. Buka folder backend
```bash
cd backend
```

2. jalankan composer install
```bash
npm install
```

3. jalankan aplikasi
```bash
npm start
```

## Install Frontend
1. Buka folder frontend
```bash
cd frontend
```

2. Install NPM
```bash
npm install
```

3. Run Aplikasi
```bash
npm run dev
```
